## 블로그를 시작합니다

안녕하세요! **DevLog.zip**에 오신 것을 환영합니다.

이 블로그는 개발하면서 배운 것들을 **압축**해서 전달하는 것을 목표로 합니다. 핵심만 빠르게 전달하되, 중요한 맥락은 놓치지 않도록 하겠습니다.

## 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

- **Next.js** — App Router 기반 풀스택 프레임워크
- **TypeScript** — 타입 안전성 확보
- **Tailwind CSS** — 유틸리티 퍼스트 스타일링
- **MDX** — Markdown + React 컴포넌트

### 코드 예시

블로그 포스트에서는 이렇게 코드를 보여줄 수 있습니다:

```typescript
interface Post {
  title: string;
  date: string;
  tags: string[];
  published: boolean;
}

function getLatestPosts(posts: Post[]): Post[] {
  return posts
    .filter((post) => post.published)
    .sort((a, b) => new Date(b.date).getTime() - new Date(a.date).getTime());
}
```

인라인 코드도 지원됩니다: `const greeting = "Hello, World!"` 처럼요.
